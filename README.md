# FAC10_study
This is the script used to analyse the firn cores and derive maps of Firn Air Content to 10 m depth (FAC10) in Greenland.

More details are available in:
Vandecrux, B., MacFerrin, M., Machguth, H., Colgan, W. T., van As, D., Heilig, A., Stevens, C. M., Charalampidis, C., Fausto, R. S., Morris, E. M., Mosley-Thompson, E., Koenig, L., Montgomery, L. E.: Firn data compilation reveals widespread decrease of firn air content in western Greenland, The Cryosphere, https://doi.org/10.5194/tc-2018-172, 2019.
